# Hướng dẫn chạy code web:

Cách 1: Chạy từ thư mục chứa code
- Bước 1: Mở thư mục chứa code của em lên (nơi có chứa file app.py, các file ảnh,...).
- Bước 2: Tại thanh địa chỉ của thư mục ở phía trên (vd: "Download > Mitsuo_web").
- Bước 3: Nhấn đúp tô đen thanh địa chỉ, xóa hết, gõ "cmd" và nhấn Enter.
- Bước 4: Trong cửa sổ cmd, gõ "py -m streamlit run app.py" > Enter
hoặc "python -m streamlit run app.py" (tùy máy)
Trình duyệt web sẽ tự động mở.

---

Cách 2: Mở / chạy bằng Visual Studio Code
- Bước 1: Click chuột phải file "app.py" > bấm chọn "Open with Code" (để mở bằng Visual Studio Code).
- Bước 2: Khi phần mềm đã mở, Trên thanh công cụ trên cùng, chọn mục "Terminal", sau đó chọn tiếp "New Terminal". Lúc này một bảng nhỏ sẽ hiện bên nửa dưới màn hình.
- Bước 3: Gõ lệnh "py -m streamlit run app.py" > Enter
Hoặc "python -m streamlit run app.py"
- Bước 4: Trình duyệt web sẽ tự động mở. 
# Trong trường hợp không tự động mở: nhấn giữ phím Ctrl, click chuột trái link http://localhost:8xxx là trang web sẽ hiện.
